OCF Virtual Web Hosting Policies:

  • The OCF will perform virtual hosting for the OCF group accounts. For virtual hosts within the berkeley.edu domain, the group account must apply to the campus IS&T for a permission to be able to host their site on the OCF web server. The OCF also hosts several non-berkeley.edu domains. However, the application process is more complicated in this case. The permission will be grated only for group accounts that can demonstrate a unique need for having a web site outside of the berkeley.edu domain. For such domains, the group account must:
    1. apply to the OCF Board Directors for a permission to host this domain.
    2. pay any and all fees and/or obtain the permission involved in obtaining and maintaining a domain name. No .com domains are allowed.
  • Virtual web sites, just like other OCF user accounts, must comply with the relevant UC Berkeley computer use policies including but not limited to the Policies and Guidelines for Publishing on the World Wide Web at the University of California, Berkeley
  • All virtual web sites hosted on the OCF must include an OCF banner on their front page that links to the OCF front page. In the case if the banner is removed, the OCF reserves the right to terminate the virtual hosting service.

Getting started:

  1. Sign up for an OCF group account if you haven't already done so.
  2. Send an email message to the IS&T DNS administrators asking for a permission on host a virtual web site on the OCF web server. Explain them the mission of your group.
  3. If IS&T approves your request, send a note to the OCF site manager asking to setup a virtual host for your group account. It should take between one and two weeks to setup a new virtual host.
  4. The files in the public_html directory of the group account will appear at the root directory of the virtual site.
  5. All mail addressed to your virtual host/domain will be forwarded to your group account. You can forward the mail sent to different addresses in your virtual host by using .forward files. For example, if you have a file named .forward+webmaster in your group account, all mail sent to secretary@virthost will be processed using the directions in this files. For example, you can put a real webmaster address(es) in the .forward+webmaster file and it will be forwarded to her. All other mail will be delivered to the group account mailbox.
  6. Right now, no separate log files are being kept for virtual hosts. However, we do collect the overall site usage statistics for the OCF which be viewed at this URL.
  7. To place the OCF Banner on your page add the following HTML code at the bottom: 
    <a href="http://www.ocf.berkeley.edu">
