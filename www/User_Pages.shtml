This is an opt-in list of homepages by OCF users. It also includes homepages of anyone connected to UC Berkeley with a homepage. The list was fairly dormant from 1996 until fall 1999. A date after a personal website link (mm/dd/yy) indicates when the site was added to this list, not when the individual (if the site is on ocf) became an ocf account holder. OCF Staff spent a few hours over several days (finished 6/29/00) going through and taking out all the dead links. So, this page is now much more useful.

If you want to be added to this page, SSH into "www.ocf.berkeley.edu" and run (if it wasn't automatically run for you) "www-optin". If you want to be removed from this page, SSH in again.

Instructions for OCF Users who want to set up homepages are here.
